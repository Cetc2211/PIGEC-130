# **App Name**: EscalaWeb

## Core Features:

- Questionnaire Selection: Allows the psychologist to select pre-existing scales or create custom evaluation plans with multiple instruments.
- Unique Link Generation: Generates a unique, shareable link for each evaluation plan. The link can be sent via WhatsApp, email, or used for in-person evaluations on any device.
- Likert Scale Response: Enables users to respond to questionnaires using Likert scale formats within the application.
- Automated Interpretation: Automatically interprets and displays results based on pre-defined rules specific to each scale.
- Secure Response Storage: Securely stores user responses in a database for future analysis and reporting.
- Automatic Report Generation: Generates reports with simple charts to visualize the evaluation results. A reasoning tool decides which chart type is most suitable for each evaluation scale.
- Multi-Device Responsiveness: Ensures the web application is fully responsive and easy to use on various devices, including mobile phones, tablets, and desktop computers.

## Style Guidelines:

- Primary color: Light pastel blue (#A7D1AB) to invoke trust, tranquility, and reliability in the sensitive evaluation content. Chosen for its gentle reassurance without being sterile or cold.
- Background color: Off-white (#F5F5F5) for a clean and neutral backdrop, ensuring readability and focus on the evaluation content.
- Accent color: Soft lavender (#E6B8B3) to provide a delicate, comforting contrast and visual interest. Lavender is associated with serenity, emotional balance, and empathy, which is fitting for an app dealing with psychology.
- Font pairing: 'Alegreya' (serif) for headlines, offering an elegant and intellectual feel, paired with 'PT Sans' (sans-serif) for body text, providing a modern and readable contrast.
- Use clear, professional icons related to evaluation and results visualization.
- Maintain a clean and intuitive layout to ensure ease of navigation and focus during questionnaire completion.
- Incorporate subtle animations for transitions and feedback to enhance user engagement without distraction.
